# Cursor harness examples

Copy into `.cursor/` at the repository root (this repo already includes live
examples in `.cursor/`).

| Asset | Harness path | Project path |
| --- | --- | --- |
| Skills | `skills/<name>/SKILL.md` | `.cursor/skills/<name>/SKILL.md` |
| Subagents | `subagents/<name>.md` | `.cursor/agents/<name>.md` |
| Hooks | `hooks/economics-hooks-example.json` | merge into `.cursor/hooks.json` |
| MCP | `mcp/mcp.json.example` | `.cursor/mcp.json` |

Official docs: [Cursor agents](https://cursor.com/docs), [Cursor hooks](https://cursor.com/docs/agent/hooks).
